KDS 31 30 35 : 2021 주요 조문
4.4.1 관 지름 기준
수직관 및 수평 지관의 크기는 해당 지역 시간최대강우량을 기준으로 한다.
4.4.2 우수 수직관
표 4.4-1 — 최대 수평투영 지붕면적 기준 DN 선정. 루프드레인 시 수직벽 면적 ½ 가산.
4.4.3 우수 수평관
표 4.4-2 — 수평 기울기 1/25 이하 우수배관 적용. 수평지관 기울기 1/100 이상 유지.
4.4.4 반원형 빗물받이
표 4.4-3 — 폭(mm) 기준 선정. DN 단위와 상이하므로 혼용 주의.
📐
수평관 기울기 보정 조건√(100/n) 보정
기울기 보정계수 dc_h 변화 결과 영향
1/100 √1.0 = 1.000 보정 없음 기준값
1/50 √2.0 = 1.414 dc ÷ 1.414 관경 ↓ 가능
1/25 √4.0 = 2.000 dc ÷ 2.000 관경 최소화
ℹ️ 원리
기울기가 급할수록 관 내 유속 증가 → 같은 단면에서 더 많은 유량 처리 가능 → 필요 단면적(dc_h) 감소 → 소구경 선정 가능. KDS 표 4.4-2는 1/100 기준이므로 다른 기울기 시 √(100/n)으로 환산.

▶ 단계별 계산식 (지붕형태 보정 포함)
A_base = A_roof × RF + A_wall × 0.5
RF : 지붕형태 면적보정계수 (평지붕=1.0 / 경사=secθ / 밸리=1.0)
A_conv = A_base × (R ÷ 100) × SF × VF
VF : 밸리형 집수집중계수 (평·경사=1.0 / 밸리=1.10~1.30)
A_drain = A_conv ÷ N
드레인당 분담 환산면적 → KDS 표 조회
📊 변수 설명
A_base : 지붕형태 보정 후 유효면적 (m²)
A_roof : 수평투영 지붕면적 (m²)
RF : 면적보정계수 — 평지붕 1.0, 경사지붕 1/cosθ (secθ), 밸리형 1.0
A_wall : 수직벽 면적 — 루프드레인 시 ½ 가산 (KDS 4.4.2)
R : 시간최대강우량 (mm/hr) — 기상청
SF : 안전 여유율 (1.00~1.30)
VF : 밸리형 집수 집중계수 (1.10/1.20/1.30)
N : 우수 드레인 설치 개수
📋 KDS 표 조회 방법
A_drain 값을 KDS 31 30 35 : 2021 표 4.4-1 (100mm/hr 열)에서 해당 값 이상인 최소 관경(DN)을 선택합니다.
수평관은 표 4.4-2, 반원형 빗물받이는 표 4.4-3 동일 적용.
💡 적용 예시
지역: 서울 (98.5mm/hr) / 면적: 500m² / 드레인: 4개 / 안전율: +20%
→ A_eff = 500m²
→ A_conv = 500 × (98.5÷100) × 1.20 = 591m²
→ A_drain = 591 ÷ 4 = 147.75m²
→ 표 4.4-1 → DN125 (215m²) 선정 ✓
