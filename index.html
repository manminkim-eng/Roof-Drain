KDS 31 30 35 : 2021 주요 조문
4.4.1 관 지름 기준
옥내·옥외 우수 수직관 및 모든 수평 지관의 크기는 해당 지역의 기상자료에 의한 시간최대강우량을 기준으로 한다.
4.4.2 우수 수직관 지름
옥내외 우수 수직관의 관 지름은 표 4.4-1의 최대 수평투영 지붕 면적으로 선정한다.
4.4.3 우수 수평관
수평 기울기 1/25 이하인 우수배관은 표 4.4-2로 선정. 수평지관 기울기는 1/100 이상.
루프드레인 수직벽 보정
루프드레인 사용 시 수직벽 면적의 ½을 수평투영 지붕 면적에 가산한다.
4.4.4 반원형 빗물받이
반원형 빗물받이 크기는 표 4.4-3의 최대 수평투영 지붕 면적 기준으로 선정.
🔢
계산식 및 설계 절차
강우량 보정 환산면적 공식
A_eff = A_roof + A_wall × 0.5
A_conv = A_eff × (R ÷ 100) × SF
A_drain = A_conv ÷ N
R: 시간최대강우량(mm/hr) · SF: 안전율 · N: 드레인 수
✅ 산정 절차
1. 시간최대강우량(R) 확인 (기상청)
2. A_eff = 수평투영면적 + 수직벽×½
3. A_conv = A_eff × R÷100 × 안전율
4. A_drain = A_conv ÷ 드레인 수
5. KDS 표 4.4-1에서 관경 선정

📊 변수 설명
A_eff : 유효 수평투영 지붕면적 (m²)
A_roof : 수평투영 지붕면적 (m²)
A_wall : 수직벽 면적 (m²) — 루프드레인 시 ½ 가산
R : 해당 지점 시간최대강우량 (mm/hr) — 기상청
SF : 안전 여유율 (예: 1.20 = +20%)
N : 우수 드레인 설치 개수
A_drain : 드레인 1개당 분담 환산면적 → KDS 표 조회
📋 KDS 표 조회 방법
A_drain 값을 KDS 31 30 35 : 2021 표 4.4-1 (100mm/hr 열)에서 해당 값 이상인 최소 관경(DN)을 선택합니다.
수평관은 표 4.4-2, 반원형 빗물받이는 표 4.4-3 동일 적용.
💡 적용 예시
지역: 서울 (98.5mm/hr) / 면적: 500m² / 드레인: 4개 / 안전율: +20%
→ A_eff = 500m²
→ A_conv = 500 × (98.5÷100) × 1.20 = 591m²
→ A_drain = 591 ÷ 4 = 147.75m²
→ 표 4.4-1 → DN125 (215m²) 선정 ✓
